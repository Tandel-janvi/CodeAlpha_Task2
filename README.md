# CodeAlpha_Task2
CodeAlpha Java Programming Internship. This is First Task "Stock Trading Platform".

The Stock Trading Simulator is a Java-based console application that allows users to simulate real-world stock trading operations.
It enables users to view market prices, buy and sell stocks, and track their portfolio performance in real-time, using an interactive text-based menu.
The program demonstrates key object-oriented programming (OOP) principles such as encapsulation, abstraction, and the use of collections (HashMap).
